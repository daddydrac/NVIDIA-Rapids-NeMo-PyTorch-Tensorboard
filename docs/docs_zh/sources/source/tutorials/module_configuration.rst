模块配置
=========

神经模块的配置可以从 YAML 文件导入，也可以导出到 YAML 文件 \
一个模块配置文件存储了创建一个实例所需要的所有参数。

.. note::
    对于可训练的神经模块，`配置` 相对于 checkpoint 是起到了互补的作用。 \
    配置文件包含了参数 (比如: 层的数量, 隐藏层的大小等)， \
    而 checkpoint 包含了实际模块的权重
