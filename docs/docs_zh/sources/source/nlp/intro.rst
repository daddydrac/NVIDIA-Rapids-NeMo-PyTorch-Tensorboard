自然语言处理
============

支持的任务和模型:

* 意图识别和槽填充
* 文本分类
* 基于面向任务的对话系统的状态跟踪
* 语言模型
* 神经机器翻译
* 问答系统
* 命名实体识别 (NER)
* 标点符号和单词大写
* GLUE 基准
* ASR 用 BERT 做后处理

所有 NLP 集合下面的例子都在 `这里 <https://github.com/NVIDIA/NeMo/tree/master/examples/nlp>`__.
